Алгоритми

Binary Search

Ефективний алгоритм для пошуку елемента в відсортованому списку

Реалізація

def binary_search(arr, x):   low = 0   high = len(arr) - 1   while low <= high:     mid = (low + high) // 2     if arr[mid] == x:       return mid     elif arr[mid] < x:       low = mid + 1     else:       high = mid - 1   return -1

Bubble Sort

Алгоритм сортування, який проходить по списку, порівнює кожну пару сусідніх елементів і при необхідності міняє їх місцями.

Реалізація:

def bubble_sort(arr):   for i in range(len(arr)):     for j in range(0, len(arr)-i-1):       if arr[j] > arr[j+1]:         arr[j], arr[j+1] = arr[j+1], arr[j]

Fibonacci

Числа Фібоначчі — це послідовність чисел, де кожне число є сумою двох попередніх чисел. Перше два числа — 0 та 1.

Реалізація:

def fibonacci(n):   if n <= 1:     return n   else:
    return fibonacci(n-1) + fibonacci(n-2)

Алгоритм Дейкстри

Алгоритм для знаходження найкоротших шляхів від джерела до всіх інших вершин в графі з не від’ємними вагами ребер.

Реалізація:

import heapq
def dijkstra(graph, start):   dist = {vertex: float('infinity') for vertex in graph}   dist[start] = 0   pq = [(0, start)]   while pq:     current_dist, vertex = heapq.heappop(pq)     if current_dist > dist[vertex]:       continue     for neighbor, weight in graph[vertex]:       distance = current_dist + weight       if distance < dist[neighbor]:         dist[neighbor] = distance         heapq.heappush(pq, (distance, neighbor))   return dist
